# 模块 3: 静态页面设计

> 使用 HTML 和 CSS 创建美观、现代的用户界面

## 学习目标

完成本模块后,你将能够:
- 设计现代、专业的 UI 布局
- 创建可复用的 CSS 组件
- 实现响应式移动设计
- 构建无障碍且用户友好的界面

## 概述

在本模块中,我们将为 Todo App Pro 创建静态用户界面。我们将专注于 HTML 结构和 CSS 样式,然后再添加 JavaScript 功能。

## 1. 设计首页布局
